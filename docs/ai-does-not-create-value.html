AI与"机器不创造价值"

> 从蒸汽机到大模型：不变资本的两百年

报告 09 / 重读马克思系列 理论 《资本论》第一卷 第6章 c/v · 第13章 机器和大工业 日期 2026-07-19
CORE_THESIS

核心论点：当AI以零边际成本"生成"一切时，最流行的断言是"机器开始创造价值了"，劳动价值论随之被宣布死亡。马克思在《资本论》第一卷第6章和第13章中给出的分析至今未被推翻：机器是不变资本，它只能把自身的价值按磨损程度转移到产品中，一分不多；新价值的唯一源泉是活劳动。AI的全部"创造力"是一种拜物教外观——它的背后站着数以百万计的数据标注员、RLHF训练师、内容审核员，以及被无偿圈占的全人类知识语料。AI没有废除劳动价值论，它只是把劳动藏得更深了。

一、引言：当机器开始"创作"

2026年，这样的场景已经稀松平常：设计师输入一句话，十秒后拿到可商用的海报；程序员的编辑器里，大半代码由AI补全；客服、翻译、短视频脚本、财报摘要，大模型以接近为零的边际成本持续产出。企业家开始部署"数字员工"，媒体讨论"无人工厂"与"AI资本主义"，悲观者预言99%的人将沦为"无用阶级"。

QUESTION

问题的提出：在这场讨论中潜伏着一个理论断言——机器也能创造价值。如果这个断言成立，马克思的政治经济学批判就整体失效了：劳动价值论失效（价值不再以劳动时间为尺度），剩余价值理论失效（利润可以来自机器而非剥削），"剥削"概念本身失效（毕竟，剥削一堆GPU算不上剥削）。

本篇要论证的是：这个断言不成立。它把使用价值的生产误当作价值的生产，把劳动生产率的提高误当作新价值的创造——这正是马克思在1867年就已逐条驳斥过的混淆，只是这一次，机器学会了说话，混淆也就显得格外逼真。

二、理论基石：第6章与第13章讲了什么

2.1 第6章：资本的两种成分

马克思把投在生产中的资本拆成性质完全不同的两部分：

"转变为生产资料即原料、辅助材料、劳动资料的那部分资本，在生产过程中并不改变自己的价值量。因此，我把它称为不变资本部分，或简称为不变资本。相反，转变为劳动力的那部分资本，在生产过程中改变自己的价值量……因此，我把它称为可变资本部分，或简称为可变资本。" —— 马克思《资本论》第一卷第6章

用公式表示，商品价值 = c（不变资本，转移的旧价值）+ v（可变资本，劳动力价值）+ m（剩余价值）。c 只是把自己已有的价值转移到产品中——转移多少，以它自身包含的劳动为限；只有 v 的使用（活劳动）才带来增殖。

WARNING

关键区分：价值与使用价值。机器能极大地提高使用价值的生产（一小时织出十倍的布、一分钟生成一百张图），但价值的实体是抽象人类劳动，机器自身的运转不添加一分人类劳动。混淆这两者，是一切"机器创造价值论"的共同根源。

2.2 第13章：机器的价值只转移，不创造

第13章把第6章的一般原理落到机器上：机器全部参加劳动过程，但只是部分地（按磨损程度）参加价值形成过程。机器越耐用、生产率越高，分摊到每件产品上的价值转移反而越少。机器使产品便宜，而不是使产品更值钱。

"像其他一切发展劳动生产力的方法一样，机器是要使商品便宜，是要缩短工人为自己花费的工作日部分，以便延长他无偿地给予资本家的工作日部分。机器是生产剩余价值的手段。" —— 马克思《资本论》第一卷第13章

注意这个措辞：机器是生产剩余价值的手段，不是剩余价值的源泉。手段通过缩短必要劳动时间、延长剩余劳动时间而起作用——剩余价值仍然全部来自工人的活劳动。

2.3 机器的资本主义使用：不为减轻劳动，而为榨取

马克思强调，机器本身可以减轻劳动、缩短工作日；但机器的资本主义使用服从的是另一个目的——增殖资本。于是第13章记录了一连串悖论：

  • 机器成了把妇女儿童纳入雇佣劳动的手段，全家人的劳动力被摊薄计价
  • 机器提高生产率，本应缩短工时，实际却延长了工作日（机器闲置对资本是纯粹损失，必须日夜运转）
  • 工作日被工厂法限制后，资本转向提高劳动强度——同样的时间里榨取更多的劳动
  • 机器排挤工人："工人阶级的一部分就这样被机器变成了过剩的人口"（呼应第6篇：产业后备军）
  • 在机器体系中，工人沦为"活的附属物"，知识与技能从工人身上转移到机器上，并作为异己的力量与工人相对立（呼应第4篇：算法资本家）

2.4 那第一个用机器的资本家为什么发财？——超额剩余价值

这是"机器创造价值论"最有力的经验依据：率先采用机器的资本家确实获得了超额利润。马克思的回答在第10章（相对剩余价值）就已给出：该资本家的商品个别价值低于社会价值，按社会价值出售，差额即超额剩余价值。

KEY

这笔超额利润不是机器创造的。它来自两个渠道：一是该资本家工人的劳动被算作"加强的劳动"、在同一小时内物化为更多价值；二是社会总剩余价值在不同资本家之间的再分配——当机器普及、社会价值被拉平，超额部分随即消失。机器转移的是价值分配的比例，不是价值的实体。

四、解剖AI：价值到底从哪里来

SERIES

本报告与系列报告的递进关系：第1篇论证劳动价值论在知识经济时代仍然成立，本篇把问题推到极限情形——当"生产"看起来完全由机器完成时；第7篇（计件工资）讲AI产业最底层的工资形式，第8篇（数据圈地）讲AI训练料的原始积累，本篇讲AI产出物的价值来源本身。

4.1 训练环节：巨额的c + 隐蔽的v

一个大模型的成本结构完美符合第6章的解剖：

  • 不变资本 c：GPU与数据中心（制造它们本身耗费大量活劳动）、电力、被圈占的训练语料（呼应第8篇：这部分c甚至是零成本掠来的）
  • 可变资本 v：算法工程师只是冰山露出水面的部分；水下的是全球南方的标注员与审核员——据报道，美国头部AI公司外包给肯尼亚的标注工时薪不足2美元，菲律宾、印度、委内瑞拉的众包工人按件计酬（每条几分钱，正是第7篇分析的最原始计件制），内容审核员以心理健康为代价过滤暴力与色情语料
# AI产业链的价值公式

模型权重  =  c(GPU+电力+语料)  +  v(工程师+标注员+审核员)  +  m
                ↑ 旧价值转移          ↑ 新价值唯一源泉

"人工智能"的人工在哪里，智能才在哪里。模型展示出的每一项"能力"，都是巨量人类劳动（付费的与无偿的）的物化。马克思说机器是"对象化劳动"对"活劳动"的支配——大模型是迄今为止规模最大的对象化劳动：全人类的文本、图像与知识被压缩进权重文件，然后反过来支配使用者。

4.2 应用环节：AI压缩的是社会必要劳动时间

AI的真正经济效应不在"创造价值"，而在压缩生产某商品的社会必要劳动时间：

  • 一张商业插画，过去需要画师8小时，现在需要"画师+AI"1小时——当这一做法普及，插画的社会必要劳动时间下降，插画作为商品的价值随之下降
  • 这不是价值被创造了，而是价值被消灭了：每件商品包含的活劳动更少了
  • 率先使用AI的资本家按旧的社会价值出售、获得超额剩余价值（2.4节的机制原样运转）；等全行业普及，超额消失，利润率回落——这就是"AI提效红利"总是短暂的原因
WARNING

一个经验迹象：如果AI真的创造价值，那么"AI含量"越高的商品应当越值钱。实际观察到的恰恰相反——AI生成内容的市场价格迅速向零靠拢。零边际成本的生产，对应的是趋近于零的活劳动含量，也就是趋近于零的价值。价格在这里罕见地诚实地反映了价值规律。

4.3 出售环节：AI公司赚的钱是什么

既然AI不创造价值，头部AI公司的巨额收入从何而来？三笔来源，全部能在马克思的框架内定位：

  1. 对应用方剩余价值的分成：企业购买API或订阅，是因为它节省的工资支出大于订阅费——本质是企业榨取的剩余价值让渡一部分给AI供应商
  2. 租金：算力与基础模型的稀缺性被少数巨头垄断，超额收费具有地租性质（呼应第3篇：算力即21世纪的土地，NVIDIA式的硬件垄断则是"租"的物质基础）
  3. 虚拟资本的膨胀：数千亿美元估值是对未来剩余价值索取权的资本化定价，与当期利润可以严重脱节（呼应第5篇：金融化与虚拟资本）

4.4 "无人车间"悖论：全自动的资本家剥削谁？

设问推到极限：如果机器不创造价值，那么一家完全自动化的工厂——没有一名工人——它的利润从何而来？这不是马克思理论的漏洞，马克思在第13章已触及这个问题（他引用尤尔的话：自动工厂主的终极目标是"取消工人"），答案分两层：

  • 个别资本层面：自动化资本按社会价值出售商品，获取的是社会总剩余价值池中的再分配份额——它"剥削"的是全社会工人阶级的总劳动。个别资本家可以没有工人，资本家阶级不能没有工人阶级
  • 体系层面：如果自动化普遍化到直接劳动趋于零，商品生产与雇佣劳动也随之瓦解——价值的尺度（劳动时间）失去了附着物，资本主义走到了自身的界限。马克思在《政治经济学批判大纲》（1857—1858年手稿）中写道：固定资本的发展表明，"一般社会知识，已经在多么大的程度上变成了直接的生产力"，届时"直接劳动时间……不可能再象以前一样，决定财富的量"
PROPHECY

这不是理论失败，而是预言。AI把这一趋势变得可见：它越是接近"全自动生产"，商品价值越是萎缩、失业越是膨胀、需求越是塌陷——资本主义在技术上实现自身的同时，在经济上瓦解自身。出路不是"更好的AI"，而是超越以雇佣劳动和价值生产为基础的社会形态。

五、超越马克思：AI的三个新特征

#新特征说明
1精神磨损以月计19世纪的机器折旧以十年计；大模型半年即被新一代取代，数千亿投资可能在产生回报前贬值——固定资本周转的空前加速，把资本推向更疯狂的应用竞赛与更短视的榨取
2原始积累与不变资本合一机器的"原料"历史上是有价的（铁矿石、棉花）；AI的"原料"是人类公共知识，通过圈占数字公地零成本获得（呼应第8篇）——不变资本的一部分直接来自剥夺
3拜物教的完成形态机器拜物教还需要蒸汽机这个庞然物质外壳；AI连外壳都可以没有——"智能"本身被物神化，劳动被藏到最深的地下：内罗毕的标注棚、贵阳的数据基地、每一个无偿"验证你是人类"的点击

六、结论

核心发现

  1. AI不创造价值，只转移价值——它是迄今为止最昂贵的不变资本。新价值的唯一源泉仍是活劳动，AI革命改变的只是劳动的组织形式与隐蔽程度，不是价值的实体
  2. AI的真正效应是压缩社会必要劳动时间——它使商品变便宜，同时通过超额剩余价值机制让先行者短暂发财；普及之后，红利消失，留下的是被排挤的劳动者（呼应第6篇）
  3. AI公司的利润是三笔旧账的新装：应用方剩余价值的分成、算力垄断的地租、未来剩余价值的资本化定价——没有一分钱来自"机器创造的价值"
  4. "无人车间"悖论不是马克思理论的反例，而是其预言的边界条件——全自动化的终点是价值生产本身的瓦解，资本主义在技术上实现自身的同时在经济上瓦解自身

出路

层面方向
制度训练语料的公共性主张：对抓取公共知识训练的模型课以"数据分红"，返还公共基金；AI供应链的劳动标准（标注员、审核员的最低工资与心理健康保障）纳入平台责任
分配AI生产率红利的劳动分享：企业因AI裁员/提效获得的收益部分进入公共基金；探讨AI红利税与全民基本服务
工时把生产率红利兑现为缩短工时而非裁员——这正是对第13章逻辑的正面使用：机器本可以减轻劳动，阻止它减轻劳动的是资本的占有形式
观念戳破"AI创造价值"的拜物教：每一次"AI生成"背后，都有无数被隐藏的人类劳动——付薪的与无偿的。看见这些劳动，是一切治理的前提
GLOBAL

国际镜鉴：美国编剧工会2023年大罢工的核心条款之一，就是限制AI对编剧劳动的替代与无偿训练使用——这是工人阶级第一次用集体谈判给"机器"划界；欧盟《AI法案》（2024年生效）要求披露训练数据摘要；多国法院正在审理的版权诉讼，实质是被圈占的知识公地要求计价（呼应第8篇）。19世纪的工人用工厂法限制机器的工作日，21世纪的工人开始用集体行动限制模型的训练集。

马克思的启示

"像其他一切发展劳动生产力的方法一样，机器是要使商品便宜，是要缩短工人为自己花费的工作日部分，以便延长他无偿地给予资本家的工作日部分。机器是生产剩余价值的手段。" —— 马克思《资本论》第一卷第13章
FINAL

最后的判断：当一块GPU以每秒数万亿次的运算生成一张图片时，我们看到的似乎是机器在创造；马克思教我们换算另一笔账——GPU里凝结着芯片工人的劳动，模型里凝结着标注员与全体网民的劳动，图片的"价值"里新增的，只有按下回车键前后那些仍然活着的人的劳动。AI时代的拜物教达到了它的完成形态：劳动第一次可以在表面上完全缺席，而恰恰因此，追问"劳动在哪里"成为这个时代最重要的理论动作。机器不创造价值——但围绕机器进行的斗争，正在创造历史。
