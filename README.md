# 重读马克思 · 系列专题报告

> 以《资本论》的方法，审视 21 世纪的资本主义新形态。
>
> "哲学家们只是用不同的方式解释世界，而问题在于改变世界。"

本仓库是一个「重读马克思」专题报告系列：回到《资本论》的理论内核——劳动价值论、剩余价值、地租理论、虚拟资本——用它们分析平台经济、996、数据垄断、金融化等当代现象。
